# SK Contractor Platform

## Project Structure
- `/backend`: Node.js + Express API
- `/frontend`: User Website (React)
- `/admin-panel`: Admin Dashboard (React)

## Local Development
1. **Backend Server:**
   ```bash
   cd backend
   npm run dev
   ```
2. **Frontend Website:**
   ```bash
   cd frontend
   npm run dev
   ```
3. **Admin Panel:**
   ```bash
   cd admin-panel
   npm run dev
   ```
